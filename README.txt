HƯỚNG DẪN SỬ DỤNG
Bước 1: Vào link https://www.tampermonkey.net/ để tải Tampermonkey
Bước 2: Trong folder mình có để file tên Verify Name Facebook v2.5.0- by Developers Official.js mở lên và tạo một trình Script mới ở Tampermonkey và dán mã của file vừa Copy vào và lưu.
Bước 3: Vào facebook, chọn vào Trang cá nhân, nhìn góc dưới bên tay phải có chữ Developers Official | Open Script Menu và bấm vào.
Bước 4: Mở file tên TickVerifyMetaFacebook.html có trong Folder và copy all mã.
Bước 5: Qua phần Developers Official | Open Script Menu sẽ có những phần như sau, các bạn Settings theo mình nhé.
1. Paste thẻ class thì dán Class bên dưới đây vào nhé
x78zum5 xdt5ytf x1wsgfga x9otpla
2. Paste mã HTML mới thì là mã HTML ở file TickVerifyMetaFacebook mà các bạn vừa Copy sẽ dán vào nhé.
3. Custom Name thì muốn hiện tên chính là gì thì các bạn nhập tên vào.
4. Custom Nickname sẽ là dạng Biệt danh, vì phiên bản cũ mình không để biệt danh nên thiếu, phiên bản này mình để biệt danh cho đầy đủ nhé. Các bạn cứ viết một biệt danh bất kì nào các bạn muốn. Phiên bản sau mình sẽ thêm phần Disable và Enable Custom Nickname cho các bạn nhé.
5. Chọn vào Always Save HTML
6. Chọn vào Enable Edit.
Sau khi xong close để tắt Menu GUI, chuột phải vào tên gốc của các bạn chọn Change để thay đổi nội dung nhé. Sẽ tự động hiện khi các bạn Refresh lại để luôn được hiển thị cách tối ưu nhất mà không phải lặp lại các bước nhé.

Tôi mới Update xong chức năng Import và Export Settings rồi đây, mấy ông có thể Copy mã json tôi để ở File Config.json rồi paste vào Import Settings ở Script để sử dụng rồi mà không phải rườm rà như cách thủ công nữa nhé.
